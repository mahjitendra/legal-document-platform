# Legal Document Platform

This is a platform for creating, managing, and sharing legal documents.

## Features

- User authentication
- Document creation and management
- Document templates
- E-signatures
- Payment processing
- Legal consultation booking

## Tech Stack

- **Frontend:** React, Redux, Axios, Tailwind CSS
- **Backend:** Python, Flask, SQLAlchemy, Celery, Redis
- **Database:** PostgreSQL
- **Deployment:** Docker, Kubernetes, Nginx

## Getting Started

### Prerequisites

- Node.js
- Python
- Docker

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your-username/legal-document-platform.git
   ```
2. Install frontend dependencies
   ```sh
   cd frontend
   npm install
   ```
3. Install backend dependencies
   ```sh
   cd backend
   pip install -r requirements.txt
   ```

### Usage

1. Start the frontend
   ```sh
   cd frontend
   npm start
   ```
2. Start the backend
   ```sh
   cd backend
   flask run
   ```
